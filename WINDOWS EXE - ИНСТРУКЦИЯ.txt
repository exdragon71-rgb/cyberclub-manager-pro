CYBERCLUB MANAGER PRO — WINDOWS EXE, ЭТАП 1

Что получится:
- переносимая папка dist\CyberClub Manager Pro;
- внутри находится CyberClub Manager Pro.exe;
- PowerShell и npm для обычного запуска больше не нужны;
- PostgreSQL всё ещё должен быть установлен и запущен;
- файл .env автоматически копируется рядом с EXE;
- EXE запускает FastAPI и открывает приложение в браузере;
- небольшое окно позволяет повторно открыть приложение или остановить сервер.

Как собрать:
1. Запустите "Собрать Windows EXE.bat".
2. Дождитесь зелёного сообщения.
3. Откройте:
   dist\CyberClub Manager Pro\CyberClub Manager Pro.exe

Почему сначала папка, а не один EXE:
папочная сборка надёжнее для первой проверки зависимостей и статических
файлов. После успешного теста можно сделать установщик Windows.
