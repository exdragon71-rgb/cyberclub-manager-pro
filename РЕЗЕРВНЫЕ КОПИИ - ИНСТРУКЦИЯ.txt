CYBERCLUB MANAGER PRO — РЕЗЕРВНЫЕ КОПИИ

1. "Создать резервную копию сейчас.bat"
   Создаёт резервную копию PostgreSQL немедленно.

2. "Включить почасовые резервные копии.bat"
   Создаёт задачу Windows, которая запускает бэкап раз в час.

3. "Выключить почасовые резервные копии.bat"
   Удаляет задачу из Планировщика Windows.

ГДЕ ХРАНЯТСЯ КОПИИ

Документы\CyberClub Manager Pro Backups\PostgreSQL

ЛОГИ

Документы\CyberClub Manager Pro Backups\Logs

ХРАНЕНИЕ

Автоматически удаляются копии и логи старше 7 дней.

БЕЗОПАСНОСТЬ

Пароль PostgreSQL не записывается в скрипты.
Он читается из backend\.env только на время выполнения pg_dump.
