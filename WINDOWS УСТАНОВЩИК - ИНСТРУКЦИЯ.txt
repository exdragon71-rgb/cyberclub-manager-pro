CYBERCLUB MANAGER PRO — WINDOWS УСТАНОВЩИК

Файлы:
- CyberClubManagerPro.iss
- scripts\build_windows_installer.ps1
- Собрать Windows установщик.bat

Сборка:
1. Запустите "Собрать Windows установщик.bat".
2. Скрипт сначала пересоберёт EXE.
3. Затем Inno Setup создаст установщик:
   installer-output\CyberClubManagerPro-Setup.exe

Установщик:
- устанавливает приложение в LocalAppData без запроса прав администратора;
- создаёт ярлык в меню «Пуск»;
- по выбору создаёт ярлык на рабочем столе;
- добавляет программу в список установленных приложений Windows;
- позволяет удалить приложение штатным способом;
- предлагает запустить приложение после установки.

ВАЖНО:
Установщик включает файл .env с реквизитами локальной PostgreSQL.
Не публикуйте и не отправляйте установщик посторонним.
Папку installer-output не добавляйте в Git.
