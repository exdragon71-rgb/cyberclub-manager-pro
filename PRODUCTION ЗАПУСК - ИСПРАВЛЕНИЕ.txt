ИСПРАВЛЕННЫЙ PRODUCTION-ЗАПУСК

Замените старый файл запуска и добавьте:

scripts\start_production.ps1
Запустить CyberClub Manager Pro production.bat

Новая версия:
- сама находит корень проекта;
- не активирует виртуальное окружение через activate.bat;
- запускает Python напрямую из backend\.venv;
- проверяет наличие npm, backend, frontend и dist;
- не закрывает окно при ошибке;
- показывает точное сообщение об ошибке;
- открывает браузер после запуска сервера.
